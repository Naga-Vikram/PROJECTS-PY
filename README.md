# Python Automation & Game Suite

A collection of Python scripts and applications ranging from daily utility automation to interactive games and real-time API integration.

## 🛠️ Project Portfolio

### 1. 📰 News Pulse (API Integration)
A command-line application that fetches real-time news articles globally.
* **Features:** Uses `requests` to connect to the Newsdata.io API, supports custom search queries, and formats timestamps for readability.
* **Concepts:** API consumption, JSON parsing, and HTTP error handling.

### 2. 🐍 Snake, Water, Gun (Game Logic)
A digital version of the classic strategy game.
* **Features:** Randomized computer moves and conditional win/loss logic.
* **Concepts:** `random` module and complex control flow.

### 3. 📄 PDF Merger (Automation)
A utility tool to combine multiple PDF documents into a single file.
* **Features:** Efficiently merges files using `pypdf` with built-in error handling for missing files.
* **Concepts:** File I/O and external library management.

### 4. 🔏 Secret Cipher (Cryptography)
A custom encoding/decoding tool that transforms text into "secret code."
* **Features:** Uses random character padding and string slicing to scramble names and messages.
* **Concepts:** String manipulation and `random.choices` for data security.

### 5. 💧 Hydration Reminder & RoboSpeaker (Windows Utilities)
* **Drinking Water Reminder:** Automates Windows 10/11 toast notifications to encourage healthy habits using `win10toast`.
* **RoboSpeaker:** A Text-to-Speech (TTS) utility using the Windows SAPI interface.
* **Concepts:** Background loops (`time.sleep`) and Windows System Integration (`win32com`).

### 6. 💰 KBC Quiz Game
A trivia game inspired by "Kaun Banega Crorepati."
* **Features:** Multiple levels with increasing prize money and a breakdown system for wrong answers.
* **Concepts:** List nesting, iterative loops, and user-input processing.

## 📊 Technical Skills Demonstrated
* **Automation:** Using Python to handle repetitive tasks like PDF merging and desktop notifications.
* **API Management:** Retrieving and cleaning external data—a vital skill for **Data Analysis**.
* **Logic & Algorithms:** Implementing game rules and encryption patterns.
* **Error Handling:** Using `try-except` blocks to ensure programs don't crash on invalid user input.
